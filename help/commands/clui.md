# CLUI 界面操作指南

RustLuaMud 使用终端字符界面（CLUI）进行交互。

---

## 界面布局

```
┌───────── 状态栏 ──────────────────────────────────────┐
│ ★ 1: mud1 (已连接)  ○ 2: mud2 (已断开)               │
├────────────────────────────────────────────────────────┤
│                                                        │
│   MUD 游戏输出区域                                       │
│   ...                                                   │
│                                                        │
├────────────────────────────────────────────────────────┤
│ > 命令输入行                                            │
└────────────────────────────────────────────────────────┘
```

- **状态栏**: 顶部，显示所有连接的状态，★ 标记当前前台连接
- **输出区域**: 中间，显示游戏输出和脚本日志
- **输入行**: 底部，输入命令

---

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+1` ~ `Alt+9` | 切换到第 1~9 个连接 |
| `Alt+0` | 切换到第 10 个连接 |
| `Alt+Left` | 切换到前一个连接（循环） |
| `Alt+Right` | 切换到后一个连接（循环） |
| 鼠标点击状态栏标签 | 切换到对应连接 |
| `Ctrl+C` / `Ctrl+D` | 退出程序 |
| `↑` / `↓` | 浏览命令历史 |
| `/load reload` | 重新加载 Lua 脚本 |

---

## 命令输入

### 直接发送到 MUD

在输入行键入命令后按回车：

```
look
north
eat apple
```

### 执行 Lua 代码

以 `/lua` 开头的命令会被作为 Lua 代码执行：

```
/lua print("Hello from Lua!")
/lua ColourNote("red", "black", "错误信息")
/lua Note("自定义消息")
```

## 多连接管理

客户端支持同时管理多个 MUD 连接：

| 操作 | 按键 / 方式 |
|------|-------------|
| 切换到连接 1 | `Alt+1` |
| 切换到连接 2 | `Alt+2` |
| 切换到连接 3 | `Alt+3` |
| 切换到连接 4 | `Alt+4` |
| 切换到连接 5 | `Alt+5` |
| 切换到连接 6 | `Alt+6` |
| 切换到连接 7 | `Alt+7` |
| 切换到连接 8 | `Alt+8` |
| 切换到连接 9 | `Alt+9` |
| 切换到连接 10 | `Alt+0` |
| 前一个连接（循环） | `Alt+Left` |
| 后一个连接（循环） | `Alt+Right` |
| 点击状态栏标签 | 鼠标左键点击标签 |
| 切换到指定连接 | `/switch <角色名\|编号>` |

当前前台连接在状态栏中以 ★ 标记。

---

## 文本复制

使用终端模拟器自带的文本选择功能复制：
- **鼠标选中**文本即自动复制到选择缓冲区（Linux 终端默认行为）
- 或使用 `Ctrl+Shift+C` 复制（GNOME Terminal、Konsole 等）
- **注意**: `Ctrl+C` 会退出客户端，请勿用于复制

---

## 脚本重载

输入 `/load reload` 可以重新加载 Lua 脚本，适用于修改脚本后快速生效。

---

## 日志文件

所有游戏输出和脚本日志会自动记录到 `logs/` 目录：

- 文件名: `<会话名>_<日期>.log`
- 每行带时间戳和分类标记
